# React + Vite + Tailwind Assignment (Starter)

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the URL shown in the terminal (usually http://localhost:5173).

## Notes
- Example API integration fetches users from jsonplaceholder.typicode.com.
- Navbar component is in `src/components/Navbar.jsx`.
- Home page lives in `src/pages/Home.jsx`.
